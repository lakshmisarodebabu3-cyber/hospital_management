Hospital Management System

Description

A web-based Hospital Management System developed using PHP and MySQL to manage hospital operations efficiently.

Features

- Patient Registration
- Doctor Management
- Appointment Scheduling
- Billing System
- User Login Authentication
- Dashboard Management

Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

Installation

Clone the Repository

git clone https://github.com/lakshmisarodebabu3-cyber/hospital_management.git

Move to XAMPP htdocs

Place the project inside:

/Applications/XAMPP/xamppfiles/htdocs/

Start MySQL

sudo /Applications/XAMPP/xamppfiles/xampp startmysql

Open Project

http://localhost/hospital_management

Database Setup

Import the SQL file into phpMyAdmin and create:

"hospital_db"
